## Overview
This repository showcases my **three virtual data internships** completed under the **Excelerate Internship Program**, focusing on real-world business challenges across **data analysis, visualization, and AI modeling**.  
Each project demonstrates a complete data workflow — from data extraction and cleaning to insights, visualization, and strategic recommendations.

## Internship 1: Data Analyst Associate Internship
**Project:** *Learner Engagement & Campaign Analytics Dashboard*  
**Tech Stack:** PostgreSQL, SQL, Looker Studio, Excel  
**Focus Areas:** ETL pipeline development, dashboard design, and campaign analytics  

### Business Impact
- Integrated **5 relational datasets** into a single master table  
- Automated reporting, reducing manual analysis by **90%**  
- Delivered insights across **52 K learners** and **146 countries**

### Key Deliverables
- ETL stored procedure with cleaning and standardization logic  
- Interactive dashboard tracking learner engagement and campaign KPIs  
- Data-driven recommendations for outreach optimization

## Internship 3: AI-Powered Data Analyst Internship
**Project:** *Student Churn Analysis & Prediction*  
**Tech Stack:** Python, Pandas, Scikit-Learn, Matplotlib, Seaborn  
**Focus Areas:** Data preprocessing, feature engineering, and predictive modeling  

###  Business Impact
- Built a **Gradient Boosting model (91% accuracy)** to predict student churn  
- Identified top churn drivers using EDA and correlation analysis  
- Supported intervention strategies for over **8,558 learners**

### Key Deliverables
- Data cleaning and feature-engineering pipeline  
- Churn prediction model with evaluation metrics  
- Insight report with retention recommendations
