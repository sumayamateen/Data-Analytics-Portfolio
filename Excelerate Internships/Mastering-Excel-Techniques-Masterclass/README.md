## Project Overview
This project was completed as part of my **Data Analytics Internship with Excelerate**.  
The goal was to analyze company sales data and design an **interactive Excel dashboard** to uncover key insights.

## Objectives
- Build a dynamic dashboard using Pivot Tables, slicers, and KPIs.
- Perform descriptive, trend, and product-level analysis.
- Present performance metrics and trends across regions and product categories.

## Tools Used
- Microsoft Excel
  - Pivot Tables
  - Slicers
  - Conditional Formatting
  - Lookup and Aggregation Functions

## Key Insights
- Identified top-performing product categories driving majority of sales.
- Detected regional sales trends and growth opportunities.
- Automated KPIs to dynamically track monthly and product-level performance.
